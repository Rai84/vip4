## 🧮 Código (blocos)

### Código em bloco (multi-linhas com destaque de sintaxe)

#### Exemplo em Java:
```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Olá, mundo!");
  }
}
```

#### Exemplo em HTML:
```html
<h1>Olá</h1>
<p>Isso é um parágrafo.</p>
```
